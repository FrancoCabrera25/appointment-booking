# Trabajo practico.

1. Clonar proyecto
2. ```npm install```
3. Crear el archivo ```.env``` y agregar las siguientes variables a ```MONGODB, PORT, JWT_SECRET```
4. Configuralas las variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```

6. Levantar: ```npm start:dev```
